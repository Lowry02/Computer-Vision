# Camera Calibration — Computer Vision Project

This repository contains the implementation and experiments for **Camera Calibration using Zhang’s Method**, developed for a Computer Vision and Pattern Recognition course.

## Contents

* Camera calibration using planar checkerboards (Zhang’s method)
* Estimation of intrinsic and extrinsic parameters
* Reprojection error analysis
* 3D object (cylinder) superimposition
* Statistical analysis of principal point uncertainty
* Comparison with ground truth poses
* Camera calibration on a custom smartphone dataset
* Nonlinear refinement via MLE (Levenberg–Marquardt)
* Radial distortion modeling and compensation
